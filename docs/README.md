# UFO Docs

This folder collects lightweight project documentation for product, engineering, and future maintenance work.

## Files

- `AGENTS.md`
  - How to work in this repo
- `TASKS.md`
  - Active and upcoming work
- `IDEAS.md`
  - Product and UX ideas worth keeping
- `CHANGELOG.md`
  - Short project-level change log
- `README.md`
  - This overview

## Why This Exists

The app is moving from feature accumulation toward integration, consistency, and quality. These docs give us one shared place to keep intent, priorities, and decisions visible.

## Suggested Usage

- Update `TASKS.md` when priorities change
- Add rough concepts to `IDEAS.md` before they become implementation work
- Record meaningful product/architecture changes in `CHANGELOG.md`
- Keep `AGENTS.md` focused on repo conventions and architectural direction

## Current Theme

The main goal is to make existing features feel like one coherent system:

- missions connected to places, notes, incidents, and lists
- clearer navigation between related objects
- better settings and access control
- stronger consistency across iOS and macOS
